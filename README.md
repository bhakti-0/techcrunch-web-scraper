# TechCrunch Web Scraper

This project demonstrates dynamic web scraping using Selenium and BeautifulSoup.
It extracts article data from TechCrunch and stores it in CSV format.

## Features

- Scrapes dynamic content
- Handles pagination via scrolling
- Modular Python code
- Logging and error handling
- CSV data storage

## Tech Stack

- Python
- Selenium
- BeautifulSoup

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the scraper:
   python techcrunch_scraper.py

## Output

- techcrunch_articles.csv
